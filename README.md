# minipaint

A small desktop painting app built with Svelte, Vite, Electron, and WebGPU.

## Requirements

- [Bun](https://bun.sh/) for package management and scripts
- A browser/GPU environment with WebGPU support

## Development

Install dependencies:

```bash
bun install
```

Run the web app in a browser:

```bash
bun run dev
```

Run the Electron desktop app in development mode:

```bash
bun run electron:dev
```

Build the production web bundle:

```bash
bun run build
```

Preview the production web bundle:

```bash
bun run preview
```

Package the Electron app:

```bash
bun run electron:build
```

## Project layout

- `src/` — Svelte application code
- `src/lib/WebGPUCanvas.svelte` — canvas UI shell and interaction wiring
- `src/lib/gpu/` — WebGPU renderer, resources, pipelines, buffers, and texture helpers
- `src/lib/document/` — layer, document, stroke history, and undo/redo logic
- `src/lib/input/` — brush, pan/zoom, keyboard, and stamp queue helpers
- `src/lib/persistence/` — project save/load, PNG, and ZIP helpers
- `src/lib/shaders/` — WGSL shaders
- `src/assets/` — bundled brush/image assets
- `electron/` — Electron main process, menu, preload, and project file IPC
- `scripts/dev.mjs` — starts Vite then launches Electron for desktop development
- `dist/` — generated production build output; do not edit manually

## Notes

- Use `bun run build` before submitting changes.
- `dist/` is generated by Vite and ignored by git.
- Local planning docs under `docs/` are ignored by git.
- WebGPU behavior should be manually verified on supported hardware after renderer or brush changes.
